Getting Started

To use this Conan registry in your projects:

  1. Add this registry as a remote: conan remote add conancrates http://localhost:8000
  2. Search for packages: conan search "*" -r=conancrates
  3. Install packages in your conanfile.txt or conanfile.py

Without Conan Client

You can also download packages directly without installing Conan:

  • Browse packages and click "Download" buttons on package pages
  • Download the manifest JSON with all package metadata and URLs
  • Use our download script for automated downloads
