ðŸ“¦ Using Packages with Conan

For Conan development workflows, use the ConanCrates CLI to download and use packages:

1. Download from ConanCrates

python conancrates.py download package/version

Downloads recipe (conanfile.py) + binary packages to ./conan_packages/package-version/

2. Add to Your Project

In your conanfile.txt:

[requires]
package/version

Or in your conanfile.py:

class MyProject(ConanFile):
    requires = "package/version"

3. Upload Packages

python conancrates.py upload package/version

ðŸ”— Direct Downloads

You can also download binaries directly from the web interface:

  • Browse to any package page
  • View available binaries for different platforms (OS, architecture, compiler)
  • Click Binary to download just the binary package
  • Click Bundle to download the package with all dependencies
  • View and download the recipe (conanfile.py) source code
